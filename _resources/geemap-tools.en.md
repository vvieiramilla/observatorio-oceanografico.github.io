    <p>
        <strong>geemap-tools</strong> is a collection of Python utilities developed to facilitate recurrent tasks in geospatial data analysis, visualization, and exploration within <em>Google Earth Engine</em>, using the <em>geemap</em> library as the main interface.
    </p>
  </div>

</div>

The tool was created in the context of academic and research activities at the
Oceanographic Observatory, with the aim of accelerating workflows,
standardizing procedures, and reducing the need for repetitive scripting in
remote sensing and environmental data analysis projects.

Main features
- Helper functions for loading and visualizing Earth Engine layers;
- Routines for rapid inspection of image collections and time series;
- Support for building interactive maps in Jupyter environments;
- Reusable structures for prototyping and teaching in geoprocessing.

Access to the code  
The source code, technical documentation, and usage examples are available on
the GitHub repository:
[https://github.com/andrebelem/geemap-tools](https://github.com/andrebelem/geemap-tools)
